

**Resend** (이메일 발송 서비스)의 도메인 설정 화면이네요. 단계별로 구체적으로 설명해드릴게요.

---

## 전체 흐름 요약

```
① Resend에 도메인 등록
  → ② DNS 레코드 추가 (도메인 관리 사이트에서)
    → ③ Resend가 검증 완료
      → ④ noreply@paxchristikorea.org 로 발송 가능
```

---

## 1단계 — Resend에 도메인 등록

화면에서 할 작업:

```
Name 필드:   paxchristikorea.org
Region:      Tokyo (ap-northeast-1)  ← 한국에서 가장 가까움, 유지
Custom Return-Path: send             ← 기본값 유지

→ [+ Add domain] 클릭
```

---

## 2단계 — DNS 레코드 추가

Add domain 클릭 후 아래 화면(DNS Records)에 레코드 목록이 나옵니다. 이걸 **도메인 관리 사이트**(가비아, AWS Route53, Cloudflare 등)에 입력해야 합니다.

Resend가 요구하는 레코드는 보통 이 3종류입니다:

| 종류 | 이름 | 값 | 용도 |
|------|------|-----|------|
| TXT | `paxchristikorea.org` | `v=spf1 include:amazonses.com ~all` | 발신 서버 인증 (SPF) |
| TXT | `resend._domainkey.paxchristikorea.org` | `p=MIGf...` (긴 키값) | 이메일 서명 (DKIM) |
| MX | `send.paxchristikorea.org` | `feedback-smtp.ap-northeast-1.amazonses.com` | 반송 메일 수신 |

> 실제 값은 Resend 화면에서 복사해야 합니다. 위 값은 예시입니다.

---

## 3단계 — 도메인 관리 사이트별 입력 방법

`paxchristikorea.org` 도메인을 어디서 관리하세요?도메인 관리 사이트를 알려주시면 해당 사이트 기준으로 정확한 입력 방법을 안내해드릴게요.

그리고 진행하면서 알아두실 점 하나 — DNS 레코드 추가 후 **전파 시간이 최대 24~48시간** 걸릴 수 있는데, 보통 한국에서는 30분~1시간 내에 완료됩니다. Resend 화면에서 Verify 버튼을 누르면 검증 상태를 실시간으로 확인할 수 있습니다.

paxchristikorea.org WHOIS 조회 내용 분석

```
Registrar: Inames Co., Ltd.   ← 도메인 구매처
Name Server: ns.gabia.co.kr   ← DNS 관리: 가비아!
             ns.gabia.net
             ns1.gabia.co.kr
```

**DNS 관리 사이트 = 가비아 (gabia.com)** 입니다.

---

## 가비아에서 DNS 레코드 추가하는 방법

### 접속 경로

```
gabia.com 로그인
→ 상단 메뉴: My 가비아
→ 왼쪽: 서비스 관리
→ DNS 관리
→ paxchristikorea.org 옆 [설정] 클릭
→ DNS 레코드 관리 탭
```

---

### Resend에서 레코드 값 복사하는 순서

Resend 화면에서 `+ Add domain` 클릭 후 DNS Records 화면이 나오면, 아래처럼 3개 레코드가 표시됩니다.

**그 값을 가비아 DNS에 이렇게 입력합니다:**

| 타입 | 호스트명 (Name) | 값 (Value) | TTL |
|------|----------------|-----------|-----|
| TXT | `@` 또는 도메인명 | Resend 화면 복사 | 3600 |
| TXT | `resend._domainkey` | Resend 화면 복사 | 3600 |
| MX | `send` | Resend 화면 복사 | 3600 |

> 가비아 입력 시 주의: 호스트명에 도메인을 **붙이지 말고** 앞부분만 입력합니다.
> 예) `resend._domainkey.paxchristikorea.org` → `resend._domainkey` 만 입력

---

## 다음 실행 순서

```
1. Resend → [+ Add domain] 클릭
2. DNS Records 화면 캡처 또는 값 복사
3. 가비아 DNS 관리에서 레코드 3개 추가
4. Resend → [Verify DNS Records] 클릭
5. 완료 후 발신 주소: noreply@paxchristikorea.org 사용 가능
```

Resend의 DNS Records 화면이 나오면 그 화면도 캡처해서 보내주세요. 가비아에 정확히 어떻게 입력할지 값 하나하나 짚어드릴게요.
