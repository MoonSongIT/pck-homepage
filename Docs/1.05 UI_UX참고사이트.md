카테고리별로 정리해드릴게요. 직접 보고 영감을 얻을 수 있는 사이트들입니다.

---

**1. 비영리 / NGO 사이트 레퍼런스**

실제 잘 만들어진 비영리 단체 사이트들입니다. PCK와 성격이 비슷한 곳들을 먼저 보세요.

- **paxchristi.net** — 팍스크리스티 인터내셔널 본부 사이트입니다. PCK의 상위 단체라 직접 참고해야 할 1순위입니다.
- **amnesty.or.kr** — 국제앰네스티 한국지부입니다. 국내 비영리 사이트 중 UI가 잘 만들어진 편입니다.
- **greenpeace.org/korea** — 그린피스 한국입니다. 임팩트 있는 Hero 섹션과 캠페인 페이지 구성이 참고할 만합니다.
- **unicef.or.kr** — 유니세프 한국입니다. 후원 페이지 UX가 특히 잘 되어 있습니다.
- **oxfam.org** — 영문이지만 국제 NGO 사이트 중 디자인 완성도가 높습니다.

---

**2. 디자인 레퍼런스 갤러리**

잘 만들어진 사이트들을 모아서 보여주는 곳들입니다.

- **awwwards.com** — 전 세계 최고 수준의 웹사이트 모음입니다. Nonprofit 카테고리로 필터링해서 보세요.
- **siteinspire.com** — 깔끔하고 감각적인 사이트들 위주로 큐레이션합니다.
- **land-book.com** — 랜딩 페이지 전문 갤러리입니다. Hero 섹션 아이디어 얻기 좋습니다.
- **godly.website** — 애니메이션과 인터랙션이 뛰어난 사이트들 모음입니다.

---

**3. UI 컴포넌트 & 패턴 참고**

특정 UI 요소(버튼, 카드, 폼 등)를 어떻게 만들지 찾을 때 씁니다.

- **ui.shadcn.com** — 이번 프로젝트에서 쓸 shadcn/ui 컴포넌트들을 직접 볼 수 있습니다. 어떤 컴포넌트가 있는지 미리 파악해두면 좋습니다.
- **tailwindui.com** — Tailwind 팀이 만든 유료 UI 컴포넌트 모음이지만 미리보기는 무료입니다. 레이아웃 아이디어 얻기 좋습니다.
- **mobbin.com** — 실제 앱과 웹의 UI 패턴 스크린샷 모음입니다. 후원 플로우, 회원가입 플로우 같은 것을 검색해서 참고하세요.

---

**4. 색상 & 디자인 토큰**

컬러 시스템 잡을 때 씁니다.

- **coolors.co** — 색상 팔레트 생성기입니다. 평화·신뢰·영성 느낌의 컬러 조합을 찾아보세요.
- **colorhunt.co** — 다른 사람들이 만든 팔레트를 구경할 수 있습니다.
- **contrast-ratio.com** — 배경색과 글자색의 명도 대비를 확인합니다. WCAG 접근성 기준 통과 여부를 체크할 수 있습니다.

---

**5. 폰트**

- **fonts.google.com** — Noto Sans KR(한글)과 Inter(영문) 조합을 미리 확인해보세요. 두 폰트를 나란히 놓고 어울리는지 보는 게 중요합니다.
- **typescale.com** — 제목·본문·캡션 폰트 크기 비율을 시각적으로 잡아볼 수 있습니다.

---

**6. 접근성 체크**

고령 사용자가 많은 가톨릭 단체 특성상 중요합니다.

- **wave.webaim.org** — URL을 입력하면 접근성 문제를 자동으로 분석해줍니다. 현재 paxchristikorea.org를 넣어보면 어떤 문제가 있는지 바로 확인됩니다.
- **webaim.org/resources/contrastchecker** — 색상 대비 검사기입니다.
