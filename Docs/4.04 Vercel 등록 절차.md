# 4-5-1: Vercel 프로젝트 생성 절차

> 최종 수정: 2026-03-25
> 배포 대상: `pck-homepage.vercel.app` (임시 도메인)

---

## 1. Vercel 로그인

- [vercel.com](https://vercel.com) 접속 → **Log In**
- **Continue with GitHub** 선택 (MoonSongIT 계정)

---

## 2. 프로젝트 생성

```
상단 [Add New...] → [Project] 클릭
```

---

## 3. Git Repository 연결

```
"Import Git Repository" 화면에서:
→ MoonSongIT/pck-homepage 찾기
→ [Import] 클릭
```

> GitHub 조직이 안 보이면 **"Adjust GitHub App Permissions"** 클릭하여 MoonSongIT 조직 접근 허용

---

## 4. Configure Project 설정

| 항목 | 값 | 비고 |
|------|-----|------|
| Project Name | `pck-homepage` | 이게 `pck-homepage.vercel.app` URL이 됨 |
| Framework Preset | `Next.js` | 자동 감지됨 |
| Root Directory | `.` (기본값) | 변경 안 함 |
| Build Command | `npm run build` | 기본값 유지 |
| Install Command | `npm ci` | 기본값 유지 |

---

## 5. Environment Variables 입력

같은 화면 하단 **"Environment Variables"** 섹션 펼치기 → 아래 13개 입력:

```
DATABASE_URL=postgresql://...        ← Supabase 연결 문자열
NEXTAUTH_SECRET=                     ← 아래 명령으로 생성
NEXTAUTH_URL=https://pck-homepage.vercel.app
NEXT_PUBLIC_SITE_URL=https://pck-homepage.vercel.app
NEXT_PUBLIC_SANITY_PROJECT_ID=       ← Sanity 프로젝트 ID
NEXT_PUBLIC_SANITY_DATASET=production
SANITY_API_TOKEN=                    ← Sanity API 토큰
NEXT_PUBLIC_TOSS_CLIENT_KEY=         ← 토스 클라이언트 키
TOSS_SECRET_KEY=                     ← 토스 시크릿 키
RESEND_API_KEY=                      ← Resend API 키
ANTHROPIC_API_KEY=                   ← Claude API 키
SUPABASE_URL=                        ← Supabase 프로젝트 URL
SUPABASE_SERVICE_KEY=                ← Supabase Service Role Key
```

### NEXTAUTH_SECRET 생성

터미널에서 실행 후 복사:

```bash
node -e "console.log(require('crypto').randomBytes(32).toString('base64'))"
```

> 각 변수 입력 시 Environment 체크박스 **Production + Preview + Development** 모두 선택
