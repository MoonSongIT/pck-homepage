## 카카오 개발자 앱 등록 가이드

### 1단계: 카카오 개발자 계정 설정 (나중에 할것임)

1. **https://developers.kakao.com** 접속
2. **로그인** (카카오 계정)
3. 상단 **내 애플리케이션** 클릭

### 2단계: 앱 생성

1. **애플리케이션 추가하기** 클릭
2. 정보 입력:
   - **앱 이름**: `팍스크리스티코리아` (또는 `PCK Homepage`)
   - **사업자명**: `팍스크리스티코리아`
   - **카테고리**: `종교/비영리`
3. **저장** 클릭

### 3단계: 키 확보

앱 생성 후 **앱 설정 → 앱 키** 페이지에서:

| 키 이름 | .env 변수 | 용도 |
|---|---|---|
| **REST API 키** | `AUTH_KAKAO_ID` | 클라이언트 ID |
| **Client Secret** | `AUTH_KAKAO_SECRET` | 아래 4단계에서 생성 |

### 4단계: Client Secret 생성

1. 좌측 메뉴 **보안** 클릭
2. **Client Secret** → **코드 생성** 클릭
3. 생성된 코드 복사
4. 상태를 **사용함**으로 변경

### 5단계: 카카오 로그인 활성화

1. 좌측 메뉴 **카카오 로그인** 클릭
2. **활성화 설정** → **ON**
3. **Redirect URI 등록**:

```
http://localhost:3000/api/auth/callback/kakao
```

> 프로덕션 배포 후 추가:
> `https://yourdomain.com/api/auth/callback/kakao`

### 6단계: 동의 항목 설정

1. 좌측 메뉴 **카카오 로그인 → 동의항목** 클릭
2. 아래 항목을 **필수 동의** 또는 **선택 동의**로 설정:

| 항목 | 설정 | 이유 |
|---|---|---|
| **닉네임** | 필수 동의 | 회원 이름 |
| **프로필 사진** | 선택 동의 | 아바타 |
| **카카오계정(이메일)** | 필수 동의 | 로그인 식별 |

> 이메일의 경우 **비즈 앱 전환**이 필요할 수 있습니다 (무료).

### 7단계: .env 설정

```env
AUTH_KAKAO_ID="여기에_REST_API_키_붙여넣기"
AUTH_KAKAO_SECRET="여기에_Client_Secret_붙여넣기"
```

현재 `.env`에 주석 처리된 부분의 `#`을 제거하고 값을 채우면 됩니다.
