## 임원진 소개 입력 절차

임원진 데이터도 **Sanity CMS**에서 관리됩니다. 페이지 경로는 `/about/team`입니다.

---

### Sanity Studio에서 등록하기

1. **Sanity Studio 접속** → `http://localhost:3000/studio`

2. **좌측 메뉴에서 "임원진"** 선택 → **"+ 새 문서"** 클릭

3. **필드 입력**

   | 필드 | 필수 | 타입 | 설명 | 예시 |
   |------|------|------|------|------|
   | `name` | O | 문자열 | 이름 | `홍길동` |
   | `role` | O | 문자열 | 직책 | `대표`, `사무국장` |
   | `bio` | X | 텍스트(4줄) | 소개글 | `2019년부터 활동...` |
   | `photo` | X | 이미지 | 프로필 사진 (120x120 표시, hotspot 지원) | — |
   | `order` | X | 숫자 | 정렬 순서 (낮을수록 먼저 표시, 기본 0) | `1`, `2`, `3` |

4. **Publish** 클릭

---

### 표시 순서 제어

`order` 필드로 **오름차순 정렬**됩니다:

```
order: 1  →  대표
order: 2  →  부대표
order: 3  →  사무국장
order: 10 →  이사
order: 20 →  감사
```

---

### 사진 없는 경우

photo를 등록하지 않으면 **이름 이니셜 아바타**가 자동 표시됩니다 (peace-navy 배경).

---

### CLI로 일괄 등록

여러 명을 한꺼번에 등록하려면:

```bash
# members.ndjson 파일 작성
{"_type": "teamMember", "name": "홍길동", "role": "대표", "bio": "창립 멤버", "order": 1}
{"_type": "teamMember", "name": "김평화", "role": "부대표", "bio": "국제 연대 담당", "order": 2}
{"_type": "teamMember", "name": "이정의", "role": "사무국장", "order": 3}

# import
npx sanity dataset import members.ndjson production
```

---

### 레이아웃 참고

- **데스크톱(lg)**: 4열 그리드
- **태블릿(md)**: 3열
- **모바일(sm)**: 2열
- 카드에 Framer Motion 스태거 애니메이션 적용

---

### 주의사항

- **Sanity 프로젝트 설정**이 먼저 필요합니다 (`.env.local`에 `SANITY_PROJECT_ID`, `SANITY_DATASET` 등)
- 페이지는 **ISR 1시간** 캐싱이므로, 즉시 반영을 원하면 재배포하거나 on-demand revalidation 설정이 필요합니다
- 데이터가 없으면 **"아직 등록된 임원진이 없습니다"** empty state가 표시됩니다
