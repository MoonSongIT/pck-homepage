## 팍스크리스티코리아 연혁 등록 방법

연혁 시스템은 **Sanity CMS**에서 데이터를 관리하도록 구현되어 있습니다. 등록 방법은 다음과 같습니다.

---

### 방법 1: Sanity Studio에서 등록 (권장)

1. **Sanity Studio 접속**
   - 개발 서버 실행 후 `/studio` 경로로 접속 (예: `http://localhost:3000/studio`)
   
2. **새 문서 생성**
   - 좌측 메뉴에서 **"연혁"** 문서 타입 선택
   - **"+ 새 문서"** 클릭

3. **필드 입력**
   | 필드 | 필수 | 설명 | 예시 |
   |------|------|------|------|
   | `year` | O | 연도 (1945~2100) | `2019` |
   | `title` | O | 제목 | `팍스크리스티코리아 창립` |
   | `description` | X | 상세 설명 | `국제 팍스크리스티 한국지부 설립...` |

4. **Publish** 버튼 클릭

페이지는 **1시간(ISR)** 후 자동 갱신되며, 최신순(연도 내림차순)으로 표시됩니다.

---

### 방법 2: Sanity CLI로 일괄 등록

여러 건을 한꺼번에 등록하려면 JSON 파일로 일괄 import할 수 있습니다.

```bash
# 1. 데이터 파일 생성 (ndjson 형식)
# history-data.ndjson 파일에 한 줄에 하나씩:
{"_type": "timeline", "year": 2019, "title": "팍스크리스티코리아 창립", "description": "국제 팍스크리스티 한국지부 설립"}
{"_type": "timeline", "year": 2020, "title": "제1회 평화포럼 개최"}
{"_type": "timeline", "year": 2021, "title": "유엔 경제사회이사회 특별협의지위 획득"}

# 2. Sanity CLI로 import
npx sanity dataset import history-data.ndjson production
```

---

### 주의사항

- **Sanity 프로젝트 설정 필요**: 메모리에 따르면 `SANITY_PROJECT_ID`가 아직 미설정 상태입니다. `.env.local`에 Sanity 프로젝트 ID와 Dataset을 먼저 설정해야 합니다.
- **연혁 페이지 경로**: `/about/history`
- **데이터가 없으면** "아직 등록된 연혁이 없습니다" empty state가 표시됩니다.
