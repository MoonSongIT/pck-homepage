## Sanity 스키마 정의 — 단계별 가이드

### 📌 선택: 스키마 관리 방식

| 방식 | 설명 | 장점 | 단점 |
|------|------|------|------|
| **(A) Sanity Studio 내장형** | Next.js 앱 안에 `/studio` 라우트로 Sanity Studio를 넣음 | 한 프로젝트에서 관리, 배포도 함께 됨 | 번들 크기 증가, 설정 좀 더 복잡 |
| **(B) 별도 Sanity 프로젝트** | `sanity init` 으로 별도 폴더에 Studio 생성 | 분리가 깔끔, 독립 배포 가능 | 2개 리포 관리 필요 |

> **추천**: 비영리 NGO 소규모 프로젝트이므로 **(A) 내장형**이 관리가 편합니다.

### Step 3-1. Sanity.io 프로젝트 생성 (웹)

1. [sanity.io/manage](https://www.sanity.io/manage) 접속 → 로그인
2. **"New Project"** 클릭
3. 프로젝트명: `pax-christi-korea`
4. Plan: **Free** (월 50만 API 요청, 충분)
5. 생성 완료 후 확인:
   - **Project ID**: 대시보드 상단에 표시 (예: `abc12xyz`)
   - **Dataset**: `production` (기본값)

---

### Step 3-2. API 토큰 생성

1. 프로젝트 대시보드 → **Settings** → **API**
2. **Tokens** 섹션 → **"Add API Token"**
   - Name: `next-app-editor`
   - Permissions: **Editor** (읽기+쓰기)
3. 생성된 토큰 복사 → 한번만 표시되므로 즉시 저장

---

### Step 3-3. CORS Origins 설정

1. 같은 API 페이지 → **CORS Origins** 섹션
2. **"Add CORS Origin"** :
   - `http://localhost:3000` (Allow credentials: ✅)
   - 나중에 프로덕션 도메인도 추가

---

### Step 3-4. `.env` 업데이트

```env
NEXT_PUBLIC_SANITY_PROJECT_ID=abc12xyz    ← 실제 Project ID
NEXT_PUBLIC_SANITY_DATASET=production
SANITY_API_TOKEN=sk_prod_xxxxxxxx         ← 실제 토큰
```

### Step 3-5. Sanity Studio 내장 (Next.js App Router)

Sanity Studio를 `/studio` 경로에 내장하려면 다음 패키지를 추가 설치합니다:

```bash
npm install sanity @sanity/vision
```
`react-simple-maps`의 React 19 peer dependency 충돌입니다 (이미 알려진 이슈). `--legacy-peer-deps` 플래그로 설치합니다.

그런 다음 아래 파일들을 생성합니다:

### Step 3-6. 스키마 파일 작성

기존 `src/types/sanity.ts` 타입과 `queries.ts` GROQ 쿼리에 **정확히 맞추어** 스키마를 정의해야 합니다:

#### ① `post.ts` — 뉴스/활동 게시글

```typescript
import { defineField, defineType } from 'sanity'

export const post = defineType({
  name: 'post',                    // GROQ: _type == "post"
  title: '뉴스/활동',
  type: 'document',
  fields: [
    defineField({
      name: 'title',
      title: '제목',
      type: 'string',
      validation: (rule) => rule.required().max(100),
    }),
    defineField({
      name: 'slug',
      title: '슬러그 (URL)',
      type: 'slug',
      options: { source: 'title', maxLength: 96 },
      validation: (rule) => rule.required(),
    }),
    defineField({
      name: 'category',
      title: '카테고리',
      type: 'string',
      options: {
        list: [
          { title: '뉴스', value: 'news' },
          { title: '활동', value: 'activity' },
          { title: '성명서', value: 'statement' },
          { title: '보도자료', value: 'press' },
        ],
      },
      validation: (rule) => rule.required(),
    }),
    defineField({
      name: 'excerpt',
      title: '발췌문',
      type: 'text',
      rows: 3,
    }),
    defineField({
      name: 'body',
      title: '본문',
      type: 'array',
      of: [
        { type: 'block' },        // Portable Text
        { type: 'image', options: { hotspot: true } },
      ],
    }),
    defineField({
      name: 'mainImage',
      title: '대표 이미지',
      type: 'image',
      options: { hotspot: true },
      fields: [
        defineField({
          name: 'alt',
          title: '대체 텍스트',
          type: 'string',
        }),
      ],
    }),
    defineField({
      name: 'publishedAt',
      title: '발행일',
      type: 'datetime',
      validation: (rule) => rule.required(),
    }),
  ],
  // Studio 미리보기 설정
  preview: {
    select: { title: 'title', media: 'mainImage', date: 'publishedAt' },
    prepare({ title, media, date }) {
      return {
        title,
        subtitle: date ? new Date(date).toLocaleDateString('ko-KR') : '',
        media,
      }
    },
  },
  // 기본 정렬
  orderings: [
    { title: '발행일 (최신순)', name: 'publishedAt', by: [{ field: 'publishedAt', direction: 'desc' }] },
  ],
})
```

#### ② `education.ts` — 평화학교 교육

```typescript
import { defineField, defineType } from 'sanity'

export const education = defineType({
  name: 'education',               // GROQ: _type == "education"
  title: '평화학교 교육',
  type: 'document',
  fields: [
    defineField({ name: 'title', title: '교육명', type: 'string', validation: (r) => r.required() }),
    defineField({ name: 'slug', title: '슬러그', type: 'slug', options: { source: 'title' }, validation: (r) => r.required() }),
    defineField({ name: 'description', title: '간략 소개', type: 'text', rows: 3 }),
    defineField({ name: 'body', title: '상세 내용', type: 'array', of: [{ type: 'block' }, { type: 'image', options: { hotspot: true } }] }),
    defineField({ name: 'startDate', title: '시작일', type: 'date', validation: (r) => r.required() }),
    defineField({ name: 'endDate', title: '종료일', type: 'date' }),
    defineField({ name: 'isRecruiting', title: '모집 중', type: 'boolean', initialValue: false }),
    defineField({
      name: 'curriculum',
      title: '커리큘럼',
      type: 'array',
      of: [{
        type: 'object',
        fields: [
          defineField({ name: 'title', title: '제목', type: 'string' }),
          defineField({ name: 'description', title: '설명', type: 'text' }),
          defineField({ name: 'date', title: '일시', type: 'string' }),
        ],
      }],
    }),
    defineField({ name: 'mainImage', title: '대표 이미지', type: 'image', options: { hotspot: true }, fields: [defineField({ name: 'alt', title: '대체 텍스트', type: 'string' })] }),
  ],
  preview: {
    select: { title: 'title', isRecruiting: 'isRecruiting' },
    prepare({ title, isRecruiting }) {
      return { title, subtitle: isRecruiting ? '🟢 모집 중' : '⚪ 마감' }
    },
  },
})
```

#### ④ `timeline.ts` — 연혁
