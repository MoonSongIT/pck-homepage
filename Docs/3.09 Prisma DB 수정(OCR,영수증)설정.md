#### 테스트 3: Supabase Storage (영수증 업로드/삭제)

**사전 준비:**
1. **Supabase 대시보드** (https://supabase.com/dashboard) 접속
2. 프로젝트 선택 (`bsccnnpebrxuinymziyn`)
3. **Settings → API** → `service_role` 키 복사
4. `.env` 파일의 `SUPABASE_SERVICE_KEY="여기에_붙여넣기"`
5. **Storage** → **New Bucket** → 이름: `receipts` → Public 체크 → 생성
6. 테스트 실행:
```bash
npx tsx scripts/test-supabase-storage.ts
```

#### 테스트 4: Claude Vision OCR (영수증 스캔)

**사전 준비:**
1. **Anthropic Console** (https://console.anthropic.com) 접속
2. **API Keys** → 새 키 생성
3. `.env` 파일의 `ANTHROPIC_API_KEY="sk-ant-..."`
4. 테스트용 영수증 이미지 1장 준비 (카페 영수증, 편의점 영수증 등)

