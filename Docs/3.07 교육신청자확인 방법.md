Supabase 대시보드 또는 Prisma Studio에서 확인할 수 있습니다.

### 방법 1: Prisma Studio (가장 빠름)

```bash
npx prisma studio
```

→ 브라우저에서 `http://localhost:5555` 자동 열림 → **EducationApplication** 테이블 클릭

### 방법 2: Supabase 대시보드

1. [https://supabase.com/dashboard](https://supabase.com/dashboard) 접속
2. 프로젝트 선택 → 좌측 **Table Editor** 클릭
3. `education_applications` 테이블 선택

두 곳 모두 이름, 이메일, 전화번호, 소속, 지원 동기, 상태(PENDING) 등을 확인할 수 있습니다.
